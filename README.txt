# Asian Alleys Studios - Website

## Hosting Instructions

1. Extract this zip file
2. Upload both files to your web server:
   - `index.html` (the main website file)
   - `logo.png` (the logo image)
3. Both files must be in the same directory
4. Access via your domain

## Files Included
- index.html - Main website with dark Japanese cyberpunk theme
- logo.png - Asian Alleys Studios logo

## Features
- Dark theme with neon red accents
- Japanese cultural design elements
- Responsive design
- Smooth animations
- Contact form
- Portfolio showcase

## Note
This is a static HTML website that can be hosted on any web server or static hosting platform (GitHub Pages, Netlify, Vercel, etc.)
